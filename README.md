# Uppgift Promises - Dashboard

## Hur köra

Jag lyckades inte få den att funka på pages så installation sker enligt följande.

-  git clone https://github.com/decimaer/Uppgift_Promises.git
-  cd Uppgift_Promises/
-  npm install
-  npm run start
